After I read this blog post it occured to me that I should really do something about my monitor.. I spend 8+ hours every day ( on average ) in front of it and it is true that it can be tiring sometimes.

I too purchased an IKEA Ledberg Led strip and an additional very small 5W Warm light CFL bulb

My monitor ( LG Flatron E2350V ) is now set on
0 brighness
70 contrast
5 sharpness
and a 6500K color temp preset

The result is eye candy! ( literally ;p )
